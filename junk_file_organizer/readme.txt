-This is a python program that organizes all your scattered files within a folder into well
organized folders named after the file types.
- for example, files with extensions .jpg, .png, etc will be put into a folder named images. 
same with other files types for videos,documents,pdfs, etc.
- to run this program, put the junkorg.py file into any folder which you want to sort and run the program. It will
sort all your files within seconds depending on number of files.
